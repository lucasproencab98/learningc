# 🧠 Treinando Lógica de Programação em C - Utilizando livro Lógica de Programação (Forbellone)

## 📌 Sobre o Repositório
Este repositório foi criado com o objetivo de desenvolver e aprimorar a **lógica de programação** utilizando a linguagem **C**.  
Aqui você encontrará exercícios, desafios e pequenos projetos que ajudam a fortalecer o raciocínio lógico e a base da programação.

---

## 🎯 Objetivos
- Melhorar o raciocínio lógico
- Praticar resolução de problemas
- Aprender e reforçar conceitos da linguagem C
- Preparação para entrevistas técnicas e provas

---

## 🛠️ Tecnologias Utilizadas
- Linguagem C
- Git
- GitHub

---


## 📚 Conteúdos Abordados
- Variáveis e tipos de dados
- Operadores
- Estruturas condicionais (if, else, switch)
- Estruturas de repetição (for, while, do while)
- Vetores e matrizes
- Funções
- Ponteiros 

---

## ▶️ Como Executar os Códigos

### 🔧 Pré-requisitos
- Ter um compilador C instalado (ex: GCC)

### 💻 Compilando e executando
gcc nome_do_arquivo.c -o programa
./programa

---

## 🚀 Como Usar Este Repositório
1. Clone o repositório:
git clone https://github.com/lucasproencab98/learningC.git


2. Acesse a pasta:
cd learningC


3. Escolha um exercício e execute
